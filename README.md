## Stack

- **Language:** Python 3
- **Environment:** Google Colab
- **Core libraries:** pandas, numpy, scikit-learn, matplotlib, seaborn, prophet

---

## Structure

Each project folder contains:
- A Jupyter notebook (`.ipynb`) with full code and outputs
- A `README.md` explaining the project, methodology, and results
